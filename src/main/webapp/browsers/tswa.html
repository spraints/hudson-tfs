Enter the full URL to a page that displays a change set on the Team System Web Access. The plugin will use the example URL as the base for building up the links to the Team System Web Access server. The plugin will strip the last path name so the URL can be used to build URL for change sets and other pages.

If the field is empty then the plugin will try to build up a valid URL using the server URL. For instance if the server URL is https://tsf02.codeplex.com then the plugin will creates links such as https://tfs02.codeplex.com/UI/Pages/Scc/ViewChangeset.aspx?changeset=14534.
